# Лабораторная работа №6-7. Взаимодействие с пользовательским интерфейсом и State в Jetpack Compose. Написание автоматических тестов
Приложение для расчёта чаевых (Tip Time), созданное в ходе лабораторной работы №6-7 позволяет пользователям быстро и удобно вычислить сумму чаевых на основе суммы счёта и выбранного процента, а также опционально округлить результат.
## Цель работы
Познакомиться с понятием состояния (state) в Android-приложениях и научиться
использовать его в интерфейсе с помощью Jetpack Compose. В результате
лабораторной работы будет создано приложение калькулятор чаевых (Tip Time),
которое автоматически рассчитывает сумму чаевых. Изучить автоматизированные
тесты в Android.
## Функциональность
Приложение предоставляет следующий набор возможностей:
-   **Ввод суммы счета:** Поле для ввода суммы, с которой будут рассчитываться чаевые. Клавиатура автоматически переключается на цифровой режим для удобства ввода.
-   **Выбор процента чаевых:** Второе текстовое поле для ввода желаемого процента чаевых.
-   **Расчет чаевых:** Автоматический пересчет итоговой суммы чаевых при изменении любого из параметров.
-   **Округление:** Переключатель (`Switch`), позволяющий округлить сумму чаевых в большую сторону.
-   **Адаптивный интерфейс:** Интерфейс поддерживает вертикальную прокрутку, что обеспечивает корректное отображение как в портретной, так и в альбомной ориентации.
-   **Наглядные подсказки:** Поля ввода содержат текстовые метки (Bill Amount, Tip Percentage) и иконки (деньги и процент), делающие интерфейс интуитивно понятным.

## Технологии
В проекте используются следующие технологии и библиотеки:
-   **UI Toolkit:** Jetpack Compose для декларативного построения интерфейса.
-   **Дизайн-система:** Material Design 3 (Material3) для стилизации компонентов.
-   **Управление состоянием:** `mutableStateOf()` и `remember()` для хранения и отслеживания состояния, а также техника "подъема состояния" (State Hoisting).
-   **Форматирование:** `NumberFormat.getCurrencyInstance()` для корректного отображения валюты.
-   **Ресурсы:** Вынос текстовых строк в `strings.xml` для упрощения локализации.
-   **Тестирование:**
    -   **Локальные тесты (Local tests):** Для проверки бизнес-логики (функции `calculateTip`).
    -   **Инструментальные тесты (Instrumentation tests):** Для проверки пользовательского интерфейса (UI тесты) с использованием правил `createComposeRule()`.
